Trithemius Cipher Challenge

The Trithemius cipher, named after the 16th-century German abbot and cryptographer Johannes Trithemius, is a polyalphabetic substitution cipher. It is an early and fundamental encryption method that is considered a precursor to more complex ciphers like the Vigen√®re cipher.

How it Works:

    Alphabet Sequence:

    • Alphabet 1: ABCDEFGHIJKLMNOPQRSTUVWXYZ
    • Alphabet 2: BCDEFGHIJKLMNOPQRSTUVWXYZA
    • Alphabet 3: CDEFGHIJKLMNOPQRSTUVWXYZAB
    • And so on...

    Encryption Process:

    • Write out the plaintext message.
    • For each letter in the plaintext, use the successive shifted alphabets to find the corresponding ciphertext letter.
    • The first letter of the plaintext is encrypted using Alphabet 1, the second letter using Alphabet 2, the third using Alphabet 3, and this pattern continues cyclically.

Example:

  • Plaintext: H E L L O
  • Ciphertext: H F N O S
