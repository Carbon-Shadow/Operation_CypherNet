Alphabetical Ranks Added Cipher Challenge

The Alphabetical Ranks Added (ARA) cipher is indeed a simple encryption method that falls under the category of substitution ciphers. Its simplicity makes it an excellent introduction to cryptography, while its basic structure can be adapted for more complex uses.

How it Works:

  1. Assign Numerical Values:
    • Each letter of the alphabet is assigned a numerical value. For example, A=1, B=2, C=3, ..., Z=26.
  2. Encrypting a Message:
    • To encrypt a message, you first convert each letter to its corresponding numerical value. Then, you add a fixed number to each value. This fixed number is often referred to as the "key" for the cipher. For example, if your key is 3, you would add 3 to each numerical value of the letters in the message.
  3. Handling Wraparounds:
    • If the addition causes the numerical value to exceed 26 (the number assigned to Z), you wrap around to the beginning of the alphabet. For instance, if the letter is X (24) and you add 3, the resulting number is 27. Since 27 exceeds 26, you wrap around and get 1 (which corresponds to A).
  4. Convert Back to Letters:
    • Once you have the new numerical values after adding the key, convert these values back to letters to get the encrypted message.

Example:

1. Convert to Numerical Values:

  • C = 3
  • R = 18
  • Y = 25
  • P = 16
  • T = 20
  • O = 15

2. Add the Key (7):

  • C: 3 + 7 = 10 → J
  • R: 18 + 7 = 25 → Y
  • Y: 25 + 7 = 32 → 32 - 26 = 6 → F
  • P: 16 + 7 = 23 → W
  • T: 20 + 7 = 27 → 27 - 26 = 1 → A
  • O: 15 + 7 = 22 → V

Encrypted Message: JYFWAV

1. Convert to Numerical Values:

  • J = 10
  • Y = 25
  • F = 6
  • W = 23
  • A = 1
  • V = 22

2. Subtract the Key (7):

  • J: 10 - 7 = 3 → C
  • Y: 25 - 7 = 18 → R
  • F: 6 - 7 = -1 → -1 + 26 = 25 → Y
  • W: 23 - 7 = 16 → P
  • A: 1 - 7 = -6 → -6 + 26 = 20 → T
  • V: 22 - 7 = 15 → O

Decrypted Message: CRYPTO
